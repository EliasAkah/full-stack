Welcome to the world of flexbox

we are going to demonstrate the effects of the different properties of flexbox. below are list of main properties of flexbox with their individual values

  • flex-container: we achieve this by setting the display property to flex or inline-flex
  • justify content: use for determining the space between the flexitem. it is also used for aligning the flex items to main axis of the container. below are it's values:
    • flex-start
    • flex-end
    • flex-center
    • space-between
    • space-around
    • space-evenly
  • flex-direction:lex-direction:gives direction of flex items within the container. it's values are: row, reverse-row, column, reverse-column
  • align-content:align multiple line flex items along the cross axis of the container. it usses the same values as the justify content property, but have major difference in the space between the flex items
  • align-items:used to align the flex items to the cross axis of the container(it is perpendicular to the main axis)
  • flex items: has propeties such as: order, flex-shrink, flex-grow; align-self, flex-basis
  • flex-wrap: determines the number of lines the flex items would occupy
